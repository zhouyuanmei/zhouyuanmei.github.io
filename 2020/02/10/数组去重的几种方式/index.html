数组去重的几种方式

发表于 2020-02-10 更新于 2020-02-16 分类于 数组去重 阅读次数： Valine：

方式一：第一种 for + indexOf()去重

思路：遍历目标数组与新数组每项作比较，若下标为-1，则表示新数组无此项，将此项push进新数组。

方式二：第二种 for + indexOf()去重

思路：遍历目标数组，若该项初次出现的下标与该项在数组的下标相同，则放入新数组。

方式三：双重for循环

思路：外层遍历目标数组，内层循环检查是否重复。若有重复则过滤前面相同的一项，若无重复则push。

方式四：ES6中的Array.filter() + indexOf()

思路：和map类似，Array的filter也接收一个函数。但是和map不同的是， filter把传入的函数依次作用于每个元素，然后根据返回值是 true 还是false决定保留还是丢弃该元素。

方法五：for……of + includes()

思路：当includes()返回false的时候，就将该元素push到空数组中

方法六：sort()

思路：sort先给数组排序，使相同的项总是相邻的，然后遍历数组和前一个对比，不相等就放入新数组中。

方法七：for + object

思路：利用对象的属性不会重复这一特性，校验数组元素是否重复。

方法八：ES6的new Set()

思路:set数据结构类似数组，但set的成员具有唯一性，可基于这一特性去重
